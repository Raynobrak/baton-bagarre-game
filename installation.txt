Pour lancer le jeu il vous suffit de suivre les étapes suivante

Vous devez avoir prealablement installé python sur votre systéme

1. Télechargez la dernière version de Baton Bagarre
2. Extraire l'archive dans le dossier de votre choix
3. Ouvrir un terminal dans le dossier extrait
4. Executez la commande suivante : python run.py
5. Le script se chargera d'installer les dépandences et de lancer le jeu
6. Il ne vous reste plus qu'a jouer et defendre le feu sacré !

Controle:

a d: deplacement du joueur
w : saut
espace : coup de poings
k : coup de pieds
q : pouvoir special
e : recharger le feu sacré

Merci d'avoir installé Baton Bagarre
